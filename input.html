mic.input.get_cli_input(cli_args)¶

Get CLI input arguments.

Parameters

cli_args – command line arguments

Returns

  • source (pathlib Path object) – source path (single stack file or directory including multiple stacks)

  • fmt (list (dtype=str)) – list of input file formats

  • dest (pathlib Path object) – output path

  • field (pathlib Path object) – path to illumination models

  • mode (int) – correction mode (see help documentation)

  • obj (str) – employed objective

  • wl (int or list of int) – channel wavelengths [nm]
