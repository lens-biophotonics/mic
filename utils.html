mic.utils.convert_to_float_and_rescale(img)¶

Optionally converts an image to float32 and rescales to [0, 1] if it is of integer type.

Parameters

image (np.ndarray) – Input image as a NumPy array.

Returns

Processed image with float32 type and values in [0, 1] if originally int;

otherwise, returns the original image unchanged.

Return type

np.ndarray

mic.utils.create_memory_map(dtype, shape=None, name='tmp', tmp=None, arr=None, mmap_mode='w+')¶

Create a memory-map to an array stored in a binary file on disk.

Parameters

  • dtype – data-type used to interpret the file contents

  • shape (tuple) – shape of the stored array

  • name (str) – optional temporary filename

  • tmp (str) – temporary file directory

  • arr (numpy.ndarray) – array to be mapped

  • mmap_mode (str) – file opening mode

Returns

mmap – memory-mapped array

Return type

NumPy memory map
